# NeuroTinder

- models.py - код для базы данных
- main.py - сервер
- gendata.py - создать профили ботов

# запуск:
```
uv sync
uv run gendata.py
uv run fastapi dev
```
Пока-что для llm используется ollama с qwen3.5:0.8b
